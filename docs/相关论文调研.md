# 方案相关论文调研

本文档整理“AI 超分辨率模型高效硬件加速器设计与实现”赛题中可支撑方案选择的相关论文、代码和阅读优先级。推荐主线为：

```text
SPAN / 高效 CNN 超分模型
  + 会议场景数据微调
  + 教师模型蒸馏
  + INT8 量化感知训练
  + FPGA 流式卷积加速器
```

## 1. 必读论文

### 1.1 SPAN：主模型首选

**Swift Parameter-free Attention Network for Efficient Super-Resolution**

- 作者：Cheng Wan, Hongyuan Yu, Zhiqi Li, Yihang Chen, Yajun Zou, Yuqing Liu, Xuanwu Yin, Kunlong Zuo
- 会议：CVPR Workshops 2024 / NTIRE
- 论文：https://arxiv.org/abs/2311.12770
- CVF：https://openaccess.thecvf.com/content/CVPR2024W/NTIRE/papers/Wan_Swift_Parameter-free_Attention_Network_for_Efficient_Super-Resolution_CVPRW_2024_paper.pdf
- 代码：https://github.com/hongyuanyu/SPAN

推荐理由：

- SPAN 是当前方案的主模型候选，目标就是高效图像超分。
- 使用 parameter-free attention，在提升重建质量的同时避免复杂 attention 结构。
- 主要由卷积、激活、残差连接和上采样组成，适合 FPGA 量化和流水线实现。
- 论文和代码均公开，便于复现、改造和答辩引用。

对本赛题的启发：

- 主实现可以基于 SPAN-Tiny / SPAN-Small 做剪裁。
- 注意力结构尽量选择无参数或低成本形式，避免 SwinIR/HAT 里的窗口 self-attention。
- 可将 SPAN 作为“高效画质模型”，再通过 INT8 QAT 和 FPGA 架构设计解决实时性。

### 1.2 NTIRE 2025 Efficient Super-Resolution：高效超分竞赛依据

**The Tenth NTIRE 2025 Efficient Super-Resolution Challenge Report**

- 会议：CVPR Workshops 2025 / NTIRE
- 论文：https://openaccess.thecvf.com/content/CVPR2025W/NTIRE/html/Ren_The_Tenth_NTIRE_2025_Efficient_Super-Resolution_Challenge_Report_CVPRW_2025_paper.html
- PDF：https://openaccess.thecvf.com/content/CVPR2025W/NTIRE/papers/Ren_The_Tenth_NTIRE_2025_Efficient_Super-Resolution_Challenge_Report_CVPRW_2025_paper.pdf

推荐理由：

- 该报告直接对应“高效超分”方向，评价维度包括运行时间、参数量、FLOPs 和 PSNR。
- 可作为方案中“为什么不选大模型、为什么选高效 CNN”的重要依据。
- 可参考其中参赛方法的模型压缩、重参数化、蒸馏和运行时优化策略。

对本赛题的启发：

- 答辩时不要只展示 PSNR，也要展示 FPS、延迟、功耗、参数量、资源占用。
- 模型选择应强调综合性价比，而不是离线画质第一。

### 1.3 NTIRE 2026 Efficient Super-Resolution：最新趋势参考

**The Eleventh NTIRE 2026 Efficient Super-Resolution Challenge Report**

- 论文：https://arxiv.org/abs/2604.03198
- 官方仓库：https://github.com/Amazingren/NTIRE2026_ESR

推荐理由：

- 这是当前最新一届 NTIRE 高效超分报告。
- 官方仓库中以 SPAN 作为 baseline，说明 SPAN 仍然是高效 SR 方向的重要基线模型。

对本赛题的启发：

- 方案可以写成“以 NTIRE 2026 ESR baseline SPAN 为基础，面向视频会议和 FPGA 部署做定制优化”。
- 可对比 2025/2026 高效 SR 方案，说明项目路线具有时效性。

## 2. 轻量超分模型基线论文

### 2.1 FSRCNN：硬件友好的早期快速 SR 基线

**Accelerating the Super-Resolution Convolutional Neural Network**

- 作者：Chao Dong, Chen Change Loy, Xiaoou Tang
- 会议：ECCV 2016
- 论文：https://arxiv.org/abs/1608.00367
- PDF：https://personal.ie.cuhk.edu.hk/~ccloy/files/eccv_2016_accelerating.pdf

推荐理由：

- FSRCNN 是经典快速超分 CNN。
- 结构简单，包括特征提取、压缩、映射、扩展和反卷积，便于硬件实现。
- 可作为软件和硬件最小闭环 baseline。

对本赛题的启发：

- 如果 SPAN 硬件实现进度受阻，可先用 FSRCNN 跑通 FPGA 端到端流程。
- FSRCNN 可作为性能下限和传统轻量模型对照。

### 2.2 ESPCN：PixelShuffle / Sub-pixel Conv 的基础论文

**Real-Time Single Image and Video Super-Resolution Using an Efficient Sub-Pixel Convolutional Neural Network**

- 作者：Wenzhe Shi 等
- 会议：CVPR 2016
- 论文：https://arxiv.org/abs/1609.05158
- CVF PDF：https://openaccess.thecvf.com/content_cvpr_2016/papers/Shi_Real-Time_Single_Image_CVPR_2016_paper.pdf

推荐理由：

- 提出了高效 sub-pixel convolution，也就是常用的 PixelShuffle 上采样思想。
- 适合解释为什么网络主体在低分辨率空间计算，最后再放大，从而降低计算量。
- 对 FPGA 实现 x2/x4 上采样模块很有参考价值。

对本赛题的启发：

- SPAN 最后上采样可采用 PixelShuffle。
- PixelShuffle 应尽量做成流式重排模块，避免单独将大尺寸中间特征写回 DDR。

### 2.3 IMDN：信息多蒸馏轻量 SR

**Lightweight Image Super-Resolution with Information Multi-distillation Network**

- 作者：Zheng Hui 等
- 论文：https://arxiv.org/abs/1909.11856
- 代码：https://github.com/Zheng222/IMDN

推荐理由：

- IMDN 是轻量 SR 中常用基线，核心是信息多蒸馏模块。
- 在轻量模型中兼顾性能和复杂度，适合作为 SPAN 之外的对比模型。

对本赛题的启发：

- 可参考其通道蒸馏思想，压缩 SPAN block 内部通道。
- 可作为消融实验对照：IMDN vs RFDN vs SPAN。

### 2.4 RFDN：残差特征蒸馏网络

**Residual Feature Distillation Network for Lightweight Image Super-Resolution**

- 作者：Jie Liu, Jie Tang, Gangshan Wu
- 论文：https://arxiv.org/abs/2009.11551
- 代码：https://github.com/njulj/RFDN

推荐理由：

- RFDN 是 AIM 2020 高效超分相关任务中的重要轻量模型。
- 引入 Feature Distillation Connection，在较低复杂度下保持较好重建质量。

对本赛题的启发：

- 可用作备选主模型或结构消融对象。
- RFDN 的 feature distillation 思想适合做轻量化解释。

## 3. 大模型与教师模型参考

### 3.1 NTIRE 2025 Image Super-Resolution x4

**NTIRE 2025 Challenge on Image Super-Resolution (x4): Methods and Results**

- 会议：CVPR Workshops 2025 / NTIRE
- 论文：https://openaccess.thecvf.com/content/CVPR2025W/NTIRE/html/Chen_NTIRE_2025_Challenge_on_Image_Super-Resolution_x4_Methods_and_Results_CVPRW_2025_paper.html
- arXiv：https://arxiv.org/abs/2504.14582
- 代码：https://github.com/zhengchen1999/NTIRE2025_ImageSR_x4

推荐理由：

- 该报告汇总了 x4 超分的强模型方案。
- 可作为 teacher model 或上限画质参考，而不建议直接 FPGA 部署。

对本赛题的启发：

- 大模型可用于知识蒸馏，让轻量 SPAN 学习高质量重建结果。
- 答辩中可说明：我们不是直接部署重模型，而是用重模型指导轻量模型训练。

### 3.2 NTIRE 2026 Image Super-Resolution x4

**NTIRE 2026 Challenge on Image Super-Resolution x4: Methods and Results**

- 论文：https://arxiv.org/abs/2604.14558
- 代码：https://github.com/zhengchen1999/NTIRE2026_ImageSR_x4

推荐理由：

- 最新 x4 图像超分挑战报告。
- 可作为最新画质上限和 teacher 候选来源。

对本赛题的启发：

- x4 的画质目标可以参考该报告，但 FPGA 主模型应继续保持轻量化。

## 4. 量化与硬件部署论文

### 4.1 INT8 量化基础

**Quantization and Training of Neural Networks for Efficient Integer-Arithmetic-Only Inference**

- 作者：Benoit Jacob 等
- 会议：CVPR 2018
- 论文：https://arxiv.org/abs/1712.05877
- Papers With Code：https://paperswithcode.com/paper/quantization-and-training-of-neural-networks

推荐理由：

- INT8 量化和 QAT 的经典论文。
- 适合作为 FPGA 使用整数推理的理论依据。

对本赛题的启发：

- 使用 fake quantization 进行量化感知训练。
- 权重和激活 INT8，累加 INT32。
- 将卷积、bias、scale、activation 融合，减少硬件开销。

### 4.2 SR 专用 FPGA 加速论文

**An Energy-Efficient FPGA-based Deconvolutional Neural Networks Accelerator for Single Image Super-Resolution**

- 论文：https://arxiv.org/abs/1801.05997

推荐理由：

- 专门讨论单图超分的 FPGA 加速器。
- 关注 SR 场景下的并行化、片上缓存和能效优化。

对本赛题的启发：

- 可借鉴其卷积/反卷积模块并行设计。
- 可作为硬件架构章节的参考论文。

### 4.3 二值化/极低比特 SR 加速参考

**Efficient FPGA Binary Neural Network Architecture for Image Super-Resolution**

- 期刊：Electronics 2024
- 论文：https://www.mdpi.com/2079-9292/13/2/266

推荐理由：

- 讨论 BNN 在图像超分 FPGA 上的应用。
- 虽然本方案推荐 INT8 而非 BNN，但该论文可作为极低比特设计的参考。

对本赛题的启发：

- BNN 速度和资源优势明显，但画质风险较高。
- 可在报告中作为“为什么不优先选择二值网络”的对比依据。

## 5. 视频会议场景相关论文

### 5.1 NTIRE 2025 视频会议质量增强

**NTIRE 2025 Challenge on Video Quality Enhancement for Video Conferencing: Datasets, Methods and Results**

- 论文：https://arxiv.org/abs/2505.18988
- PDF：https://www.microsoft.com/en-us/research/wp-content/uploads/2025/04/Video_Quality_Enhancement_Challenge_NTIRE_CVPR_2025.pdf

推荐理由：

- 直接面向视频会议质量增强场景。
- 涉及视频会议中的压缩、模糊、降噪、超分和主观质量问题。

对本赛题的启发：

- 训练数据应加入低码率压缩、会议人像、屏幕共享、文档文字等退化。
- 评价指标不应只看整图 PSNR，还应关注人脸和文字区域。

## 6. 推荐阅读顺序

第一优先级：

1. SPAN：Swift Parameter-free Attention Network for Efficient Super-Resolution
2. NTIRE 2025 Efficient Super-Resolution Challenge Report
3. ESPCN：Real-Time Single Image and Video Super-Resolution
4. Quantization and Training of Neural Networks for Efficient Integer-Arithmetic-Only Inference

第二优先级：

1. FSRCNN：Accelerating the Super-Resolution Convolutional Neural Network
2. RFDN：Residual Feature Distillation Network
3. IMDN：Information Multi-distillation Network
4. SR FPGA accelerator：Energy-Efficient FPGA-based Deconvolutional Neural Networks Accelerator

第三优先级：

1. NTIRE 2025 / 2026 Image SR x4 报告
2. NTIRE 2025 Video Conferencing Quality Enhancement 报告
3. BNN / 极低比特 SR FPGA 论文

## 7. 对本项目的最终引用组合

建议最终设计报告至少引用以下 8 篇：

| 用途 | 推荐引用 |
|---|---|
| 主模型依据 | SPAN, CVPRW 2024 |
| 高效 SR 竞赛依据 | NTIRE 2025 Efficient SR Report |
| 最新趋势 | NTIRE 2026 Efficient SR Report |
| 上采样模块依据 | ESPCN, CVPR 2016 |
| 快速 SR baseline | FSRCNN, ECCV 2016 |
| 轻量模型对比 | IMDN 或 RFDN |
| INT8 量化依据 | Jacob et al., CVPR 2018 |
| 硬件加速依据 | FPGA-based Deconvolutional NN Accelerator for SISR |

推荐最终技术表述：

```text
本项目以 SPAN 为主干网络，参考 NTIRE Efficient Super-Resolution 系列挑战的高效模型设计思想，
采用 ESPCN 的 sub-pixel convolution 作为上采样模块，并结合 INT8 量化感知训练与 FPGA 流式卷积阵列，
实现面向视频会议场景的 x2/x4 端侧实时超分。
```
